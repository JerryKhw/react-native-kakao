# iOS Configuration

:::info
Skip this document if you are using Expo.
:::

## Register KakaoTalk App Scheme

Add `kakaoplus` to `LSApplicationQueriesSchemes` in `Info.plist`.

:::tip
By default, include `kakaokompassauth`, `kakaolink`, and `kakaoplus`.
:::

[Official Documentation](https://developers.kakao.com/docs/latest/ko/ios/getting-started#project-plist)
