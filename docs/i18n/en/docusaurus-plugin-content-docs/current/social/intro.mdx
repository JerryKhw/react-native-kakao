# Getting Started with KakaoTalk, Social, Friends

## Before You Begin

Make sure to install the dependencies for the `Core` package by referring to the [official installation guide](/docs/install-core). Also, complete the [platform-specific configurations](/docs/install-android).

## Getting Started

The `social` package allows you to retrieve KakaoTalk profiles and fetch friends list.

Internally, it utilizes the Kakao SDK's `friend` and `talk` SDKs.

Refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/kakaotalk-social/common) for more detailed information.

## Installation

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```shell
    npm install --save @react-native-kakao/user @react-native-kakao/social
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```shell
    yarn add @react-native-kakao/{user, social}
    ```
  </TabItem>
  <TabItem value="expo" label="Expo">
    ```shell
    npx expo install @react-native-kakao/user @react-native-kakao/social
    ```
  </TabItem>
</Tabs>
