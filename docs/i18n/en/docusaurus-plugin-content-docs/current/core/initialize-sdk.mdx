# Initialize the SDK

## Initialize the SDK

Call the `initializeKakaoSDK` function in `index.js` or `App.tsx`, or similar, before any Kakao SDK is used after the app starts.

```tsx title='index.js'
import { initializeKakaoSDK } from '@react-native-kakao/core';

initializeKakaoSDK('{{ native app key }}');
```

:::info
This function must always be called first, before any Kakao API is invoked.
:::

## Web Platform

If supporting the Web, you must also pass the javascript key and rest api key with the `web` option.

```ts
import { initializeKakaoSDK } from '@react-native-kakao/core';

Core.initializeKakaoSDK('{{ native app key }}', {
  web: {
    javascriptKey: '{{ javascript key }}',
    restApiKey: '{{ rest api key }}',
  },
});
```
