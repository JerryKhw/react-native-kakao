# Expo Configuration

## iOS - Register Scheme for Kakao Navigation App Launch

Add the following to `app.json`:

```json title='app.json'
[
  "@react-native-kakao/core",
  {
    "nativeAppKey": "{{native app key}}",
    "ios": {
// highlight-next-line
      "naviApplicationQuerySchemes": true
    }
  }
],
```
