# iOS Configuration

:::info
Skip this document if you are using Expo.
:::

## Register Scheme for Kakao Navigation App Launch

Add `kakaonavi-sdk` to `LSApplicationQueriesSchemes` in `Info.plist`.

![figure](https://raw.githubusercontent.com/mym0404/image-archive/master/202404242110641.webp)

:::note
For further details, refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/kakaonavi/ios).
:::
