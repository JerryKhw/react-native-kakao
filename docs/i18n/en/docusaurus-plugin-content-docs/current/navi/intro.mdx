# Getting Started with Kakao Navigation

## Before You Begin

Make sure to install the dependencies for the `Core` package by referring to the [official installation guide](/docs/install-core). Also, complete the [platform-specific configurations](/docs/install-android).

## Getting Started

The `navi` package allows you to utilize the Kakao Navigation feature.

Kakao Navigation provides SDK and API for navigation functions in the service. You can simply use the Kakao Navigation application to access navigation features or implement navigation and route information usage within the service app.

Refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/kakaonavi/common) for more details.
