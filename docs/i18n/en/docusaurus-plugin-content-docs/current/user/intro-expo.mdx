# Expo Configuration

## Android, iOS Setup

Add the following to `app.json`.

```json title='app.json'
[
  "@react-native-kakao/core",
  {
    "nativeAppKey": "{{native app key}}",
    "android": {
      "authCodeHandlerActivity": true
    },
    "ios": {
      "handleKakaoOpenUrl": true
    }
  }
],
```
