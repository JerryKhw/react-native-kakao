# Getting Started with Kakao User

## Before You Begin

You should [install the dependencies](/docs/install-core) for the `core` package and complete the related [platform-specific setup](/docs/install-android).

## Getting Started

The `user` package contains functionality related to Kakao login and user management.

Please refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/kakaologin/common).

## Installation

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```shell
    npm install --save @react-native-kakao/user
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```shell
    yarn add @react-native-kakao/user
    ```
  </TabItem>
  <TabItem value="expo" label="Expo">
    ```shell
    npx expo install @react-native-kakao/user
    ```
  </TabItem>
</Tabs>

## Console Setup

Before implementing Kakao login and user API, additional settings are required in the Kakao Developer Console.

Please refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/kakaologin/prerequisite).
