# Getting Started with Kakao Share

## Before You Begin

Make sure to install the dependencies for the `core` package by referring to the [official documentation](/docs/install-core). Also, complete the [platform-specific configurations](/docs/install-android).

## Getting Started

The `share` package contains functions related to Kakao login and other related features.

Refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/message/common) for more details.

![image](https://raw.githubusercontent.com/mym0404/image-archive/master/202404271646217.webp)

:::tip
The image above demonstrates a situation where sharing can be done using a web browser in environments where KakaoTalk is not installed. If KakaoTalk is installed on the device, sharing will work through KakaoTalk.
:::

## Installation

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```shell
    npm install --save @react-native-kakao/share @react-native-kakao/social
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```shell
    yarn add @react-native-kakao/{share,social}
    ```
  </TabItem>
  <TabItem value="expo" label="Expo">
    ```shell
    npx expo install @react-native-kakao/share @react-native-kakao/social
    ```
  </TabItem>
</Tabs>

## Configure the Console

Before implementing Kakao share and messaging features, additional settings need to be configured in the Kakao developer console.

Refer to the [Official Documentation](https://developers.kakao.com/docs/latest/ko/message/prerequisite) for more information.
