# Expo 설정

## Expo 설정

Expo는 간편하게 미리 구성된 [config plugin](https://docs.expo.dev/config-plugins/introduction/)를 사용할 수 있습니다.

:::tip
해당 패키지에서 제공하는 기능 외에 config plugin의 구성요소가 더 필요하다면
[Android 설정](/docs/install-android)과 [iOS 설정](/docs/install-ios)를 참고하여 직접 작성해 사용할 수 있습니다.
:::

## expo-build-properties 설치

```shell
npx expo install expo-build-properties
```

## app.json 설정

처음에 설정할 것은 많지 않지만, 다른 패키지들을 사용하면서 `android`와 `ios`에 필요한 설정을 추가하는 방식입니다.

```json title='app.json'
"plugins": [
  [
    "expo-build-properties",
    {
      "android": {
        "extraMavenRepos": [
          "https://devrepo.kakao.com/nexus/content/groups/public/"
        ]
      }
    }
  ],
  [
    "@react-native-kakao/core",
    {
      "nativeAppKey": "{{native app key}}",
      "android": {
        ...
      },
      "ios": {
        ...
      }
    }
  ]
]
```

## (Optional) Android 난독화 설정

[Android 설정](/docs/install-android)를 참고하여 [expo-build-properties](https://docs.expo.dev/versions/latest/sdk/build-properties/)
config plugin으로 proguard rules를 주입해줄 수 있습니다.

## (Optional) Web 플랫폼 지원

[Web 설정](/docs/install-web)을 참고하여 동일하게 진행되어야 합니다.
