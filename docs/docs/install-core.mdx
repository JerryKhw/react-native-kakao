# 공통 패키지 설치

모노레포의 구성이며 모든 패키지는 `@react-native-kakao/core` 패키지에 의존성을 가집니다.

다음과 같이 설치할 수 있습니다.

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```shell
    npm install --save @react-native-kakao/core
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```shell
    yarn add @react-native-kakao/core
    ```
  </TabItem>
  <TabItem value="expo" label="Expo">
    ```shell
    npx expo install @react-native-kakao/core
    ```
  </TabItem>
</Tabs>

다른 패키지들은 `core` 패키지에 peer dependency를 가지며 이는 `@react-native-kakao/core`는 항상
같이 설치가 되어야 함을 의미합니다.
