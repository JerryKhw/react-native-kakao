# 시작하기
## 시작하기

모든 패키지는 `@react-native-kakao/core` 패키지에 의존성을 가집니다.

다음과 같이 설치할 수 있습니다.

<Tabs>
  <TabItem value="npm" label="NPM" default>
    ```shell
    npm install --save @react-native-kakao/core
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```shell
    yarn add @react-native-kakao/core
    ```
  </TabItem>
  <TabItem value="expo" label="Expo">
    ```shell
    npx expo install @react-native-kakao/core
    ```
  </TabItem>
</Tabs>

## SDK 초기화하기

`index.js`나 `App.tsx`등 앱이 실행되고 Kakao SDK를 어떤 것도 쓰기 전에 `initializeKakaoSDK` 함수를 호출해줍니다.

```tsx title='index.js'
import { initializeKakaoSDK } from '@react-native-kakao/core';

initializeKakaoSDK('{{ native app key }}');
```
