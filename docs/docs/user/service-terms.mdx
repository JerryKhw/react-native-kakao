# 서비스 약관 동의

## 서비스 약관 동의 내역 얻어오기

[공식 문서](https://developers.kakao.com/docs/latest/ko/kakaologin/ios#terms)

`serviceTerms()`로 서비스 약관 동의 내역을 얻어올 수 있습니다.

:::warning
카카오 싱크를 사용하는 앱만 사용 가능합니다.
:::

응답은 다음과 같습니다.

```tsx
Promise<{
  allowedServiceTerms?: KakaoServiceTerms[];
  appServiceTerms?: KakaoAppServiceTerms[];
}>
```

다음은 해당하는 타입들입니다.

```tsx
export type KakaoServiceTerms = {
  tag: string;
  agreedAt: number;
};
export type KakaoAppServiceTerms = {
  tag: string;
  createdAt: number;
  updatedAt: number;
};
```

### 예시

```tsx
serviceTerms()
```
